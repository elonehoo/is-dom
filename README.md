# Is Dom

Check if the given object is a dom node.

## Install

```bash
# npm
npm install --save-dev @elonehoo/is-dom

# yarn
yarn add --dev @elonehoo/is-dom

# pnpm
pnpm install --save-dev @elonehoo/is-dom
```

## Usage

```typescript
import isDom from '@elonehoo/is-dom'

isDom(window.document)
```
